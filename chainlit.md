# Welcome to Study Partner! 📚

Your AI-powered study companion that helps you learn **anything** — from data science to Japanese.

## What I Can Do

| Feature | Description |
|:---|:---|
| 🔍 **Find Resources** | Courses, books, videos, tutorials — prioritizing free and community-recommended, all with verified links |
| 📅 **Create Study Plans** | Structured roadmaps with week-by-week timelines, milestones, and self-assessment checkpoints |
| 🎓 **Explain Concepts** | Answer questions with supporting sources and citations |
| 🧠 **Quiz You** | Interactive multiple-choice quizzes to reinforce learning through active recall |

## Quick Start Examples

- *"Find me free resources to learn Python for data science"*
- *"Create a 3-month study plan for machine learning"*
- *"Explain what neural networks are in simple terms"*
- *"Quiz me on SQL basics"*
- *"Test my understanding of gradient descent"*

## How It Works

I search the web (including Reddit) for the most up-to-date, community-vetted resources. Every link I share is verified against actual search results — I never make up URLs. When I find relevant Reddit discussions, I'll link directly to those threads too.

After learning a topic, ask me to quiz you — active recall is one of the most effective study techniques!

Built with ❤️ using LangGraph, Groq, and open-source models.
