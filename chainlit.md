# Welcome to Study Partner! 📚

Your AI-powered study companion that helps you learn **anything** — from data science to Japanese.

## What I Can Do

| Feature | Description |
|:---|:---|
| 🔍 **Find Resources** | Courses, books, videos, tutorials — prioritizing free and community-recommended |
| 📅 **Create Study Plans** | Structured roadmaps with week-by-week timelines and milestones |
| 🎓 **Teach & Explain** | Answer questions and explain concepts like a patient professor |
| 👥 **Recommend Experts** | Professors, YouTube creators, and authors to follow |

## Quick Start Examples

- *"Find me free resources to learn Python for data science"*
- *"Create a 3-month study plan for machine learning"*
- *"Explain what neural networks are in simple terms"*
- *"Help me learn Japanese from scratch"*
- *"Who are the best professors to follow for statistics?"*

## How It Works

I search the web (including Reddit) for the most up-to-date, community-vetted resources. Every link I share is verified against actual search results — I never make up URLs.

Built with ❤️ using LangGraph, Groq, and open-source models.
